# SmartHome AI Planner (智装侠)

欢迎使用 SmartHome AI Planner！这是一个创新的 Web 应用，旨在利用人工智能（AI）帮助用户轻松规划和定制他们的智能家居方案。无论您是技术小白还是DIY爱好者，只需简单输入您的居住信息和个性化需求，我们的 AI 顾问就能为您量身打造一份详细、专业且符合预算的智能家居产品采购清单和分析报告。

## 项目核心功能

本应用的核心是通过一个智能、交互式的流程，将用户的模糊需求转化为具体、可执行的智能家居方案。

### 1. 智能规划表单 (Interactive Planning Form)

- **引导式信息输入**: 用户通过一个清晰、分步的表单来提供基础信息，包括：
    - **房产信息**: 房屋面积和户型结构。
    - **预算等级**: 提供“经济”、“高级”、“豪华”三个档位，满足不同用户的消费能力。
    - **个性化场景标签**: 为了让用户更轻松地表达需求，我们设计了场景化标签，取代了宽泛的文本输入：
        - **家庭成员结构**: 如“独居青年”、“有娃家庭”、“家有长辈”等，帮助AI理解核心居住成员的特点。
        - **核心关注体验**: 如“全屋安防”、“影音娱乐”、“懒人自动”等，让AI聚焦于用户最看重的智能体验。
    - **户型图上传 (可选)**: 用户可以上传户型图照片，AI 将结合图像分析，提供更精准的空间设备布局建议。
    - **补充需求**: 保留一个文本框，供用户输入更具体或特殊的定制需求。

### 2. AI 驱动的方案生成 (AI-Powered Proposal Generation)

- **深度需求分析**: 应用的后端利用 GenAI/LLM 模型，将用户在表单中输入的所有信息（包括从场景标签转换而来的自然语言上下文）与我们的产品数据库进行综合分析。
- **智能产品筛选**: AI 会根据用户的预算、户型、家庭成员特点和核心关注点，从丰富的产品库中筛选出最匹配的智能家居产品组合。
- **生成分析报告**: 除了产品清单，AI 还会生成一份专业的顾问分析报告，内容包括：
    - **方案价值**: 阐述该方案实现了哪些核心的自动化功能。
    - **预算权衡**: 解释因预算限制，哪些功能或产品体验上做出了妥协。
    - **升级建议**: 如果预算充足，可以从哪些方面进行升级。
    - **省钱技巧**: 提供一些可以降低成本的替代方案或技巧。

### 3. 动态结果展示页 (Dynamic Results Dashboard)

- **清晰的方案呈现**: 生成的方案会以一个美观、易读的仪表盘形式呈现给用户。
- **分类产品清单**: 所有推荐的产品会按“客厅”、“卧室”等房间进行分类，并以表格形式清晰列出产品图片、名称、品牌、数量、单价和小计。
- **总预算与 AI 分析**: 页面右侧会固定显示方案的总预算和 AI 生成的顾问分析报告。
- **“解锁查看”功能**: 为了有效收集潜在客户信息，除“客厅”外的其他区域清单以及总价和分析报告默认被“毛玻璃”效果遮挡。用户只需输入姓名和手机号即可解锁查看完整方案，极大地提升了线索转化率。

### 4. 响应式与多语言设计

- **完全响应式**: 应用界面在桌面和移动设备上均有良好的视觉和交互体验。
- **国际化支持**: 内置了中、英、日、韩四种语言，可以根据浏览器语言自动切换，满足不同地区用户的需求。

## 技术栈

- **前端**: Next.js (App Router), React, TypeScript
- **UI 组件库**: Shadcn/UI, Tailwind CSS
- **图标**: Lucide React
- **表单处理**: React Hook Form, Zod (用于数据验证)
- **AI 与后端**: Genkit / OpenAI-compatible API, Next.js Server Actions
- **状态管理**: React Context API
- **国际化 (i18n)**: 自定义 React Context 实现
